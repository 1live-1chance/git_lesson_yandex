# Coffee Information Application

Приложение для отображения информации о кофе из базы данных SQLite.

## Описание

Простое приложение на PyQt6, которое подключается к базе данных SQLite и отображает информацию о различных сортах
кофе в табличном виде.

## Функциональность

- Отображение данных о кофе в таблице
- Просмотр информации: ID, название сорта, степень обжарки, тип помола, описание вкуса, цена, объем упаковки
- Интерфейс на основе Qt Designer

## Требования

- Python 3.6+
- PyQt6
- SQLite3 (входит в стандартную библиотеку Python)

## Установка

1. Клонируйте репозиторий:
   ```
   git clone <url-репозитория>
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Использование

Запустите приложение:
```
python main.py
```

## Структура проекта

- `main.py` - основной файл приложения
- `main.ui` - файл интерфейса Qt Designer
- `coffee.sqlite` - база данных SQLite с информацией о кофе
- `requirements.txt` - список зависимостей
